# 演示账号说明

## 当前登录定位

ExpoPilot OS 当前登录页用于演示入口，不是生产级账号系统。

| 项目 | 当前状态 |
|---|---|
| 账号体系 | 前端 demo 校验 |
| 沙盒快速登录 | 支持 |
| 角色切换 | 支持运营管理员 / 普通成员 |
| 真实用户注册 | 不支持 |
| 真实密码找回 | 不支持 |
| 真实业务数据 | 不接入 |

## 对外说明话术

> 当前账号用于展会演示。系统没有接入真实账户、真实个人信息和生产数据；沙盒登录只用于切换演示角色。后续如果进入真实部署，需要接入正式身份认证、权限控制、审计留痕和数据隔离。

## 安全边界

| 边界 | 说明 |
|---|---|
| 不保存真实密码 | 演示环境不处理真实账户 |
| 不接真实后端 | 所有任务状态为 demo 数据或本地状态 |
| 不让 Agent 自动执行 | Agent 只解释和建议，项目经理保留确认权 |
| OpenClaw 只做解释来源 | 不作为主决策源或执行器 |

## 评委可能追问

| 问题 | 回答 |
|---|---|
| 这个登录安全吗？ | 当前是演示登录，不是生产认证。真实部署会接入正式身份认证和权限系统。 |
| 数据是真的吗？ | 当前使用 demo 数据，目的是展示异常处置流程和责任记录方式。 |
| Agent 能自动派单吗？ | 不能。DispatchAgent 只给建议，必须由项目经理确认。 |
| OpenClaw 负责决策吗？ | 不负责。OpenClaw 只提供解释文本。 |
