# ExpoPilot OS 试点走查脚本

## 1. 登录

- 以主办方身份登录，进入项目首页。
- 说明这是现场运营智能体系统，不是传统会展 SaaS。

## 2. 项目首页

- 打开 `2026 未来展现场运营`。
- 先看项目摘要：区域数、数据源、待处理事件、未闭环任务。

## 3. 现场配置

- 说明项目运行前必须完成区域和数据源配置。
- 展示视觉输入、人工补录输入和降级切换入口。

## 4. 实时态势

- 查看顶部四个核心指标。
- 聚焦当前最高优先级事件。
- 点击某个区域看输入模式、健康状态和推荐动作。

## 5. 事件中心

- 展示系统事件和人工补录事件。
- 操作确认、忽略、升级和查看解释。

## 6. 任务调度

- 对关键事件执行推荐派发。
- 展示改派、撤回和回执状态。

## 7. 工作人员端

- 切换到工作人员身份。
- 操作 `已接收 / 已完成 / 无法执行 / 需要支援` 四种反馈。

## 8. 复盘分析

- 查看结论摘要、关键指标、关键事件与审计日志。
- 操作从事件沉淀为策略和导出复盘摘要。

## 9. 策略库

- 展示系统模板和复盘沉淀策略。
- 说明后续可作为智能体学习与复用入口。
